# 🎛️ Ableton Remote Script — MCP Server Integration

This Remote Script provides a communication bridge between **Ableton Live** and the **MCP Server**, enabling extended control surface features, advanced parameter access, and deeper integration with external hardware such as the **X-Touch Mini / X-Touch**.

## 🚀 Version Overview

This version of the Remote Script introduces improved stability, cleaner communication with the MCP Server, and expanded feature support.

### 🔧 Key Features
- **Bi‑directional communication** with the MCP Server  
- **Automatic detection** of the server and Live session state  
- **Clip creation support** in Arrangement View  
- **Access to plugin parameters** through the MCP protocol  
- **Improved reliability** and reduced latency  
- **Better error handling** when the server is unavailable  

## 🧩 Requirements
- Ableton Live (tested with Live 11/12)  
- MCP Server (latest version recommended)  
- Compatible control surface (e.g., X‑Touch Mini / X‑Touch)

## 📡 How It Works
The Remote Script sends and receives MCP‑formatted messages to the server, allowing Ableton to expose functions normally unavailable through standard MIDI Remote Scripts.  
This enables deeper integration, more responsive controls, and new creative workflows.
